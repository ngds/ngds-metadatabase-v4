

Service Start up for the front end httpd application

/etc/systemd/system/ngds-app.service
 
[Unit]
Description=NGDS Search HTTP Server
After=network.target

[Service]

User=root
Group=root
Restart=always
RestartSec=500ms
StartLimitInterval=0
KillSignal=SIGQUIT
WorkingDirectory=/home/ngds-admin/NodeJs
ExecStart=/usr/bin/node /home/ngds-admin/NodeJs/ngds-app.js

[Install]
WantedBy=multi-user.target



Commands

$ sudo systemctl daemon-reload
$ sudo systemctl enable ngds-app
$ sudo systemctl start ngds-app
